# Solar PV Installation, Operation and Maintenance

## Overview

This repository showcases project work completed during the Solar Power Installation, Operation and Maintenance Internship conducted at NIELIT Agartala.

The project covers solar photovoltaic (PV) fundamentals, system design, installation planning, operation, maintenance practices, and PVsyst-based performance simulations for both grid-connected and standalone solar power systems.

## Project Components

### 1. Solar PV Internship Presentation

A comprehensive presentation covering:

* Solar energy fundamentals
* Types of solar panels
* Solar PV system components
* Installation procedures
* Grid-connected and off-grid systems
* IEC standards
* Cost analysis and government subsidies
* Maintenance and safety practices
* Future trends in solar energy

### 2. Grid-Connected PV System Design

A PVsyst simulation of a 61.2 kWp grid-connected solar power plant.

Key Results:

* Installed Capacity: 61.2 kWp
* Annual Energy Production: 81,889 kWh/year
* Specific Yield: 1,338 kWh/kWp/year
* Performance Ratio: 83.78%

### 3. Standalone (Off-Grid) PV System Design

A PVsyst simulation of a standalone solar system with battery storage designed for residential energy requirements.

Key Results:

* Installed Capacity: 2.6 kWp
* Average Load Requirement: 8.6 kWh/day
* Solar Fraction: 98.19%
* Useful Solar Energy: 3,065.74 kWh/year
